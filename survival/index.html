Server Address: survival.bjornie.dev
Bedrock Port: 19132 (should be default port)

Java Players

You have it easy. To connect to the server on Java Edition versions 1.9-1.21.1 you can use the address above. You will need to be whitelisted, so request to be whitelisted with your username to join in my direct messages. Sometimes I mishandle whitelist usernames as data, so in a next season you may be removed from the whitelist, in which cause you can directly message me to get it fixed.

Bedrock Players

Bedrock users on PC, phone/tablet can use the Ip and port to add Bjornie Survival to their server list. Make sure you are on the latest version of Minecraft.
Bedrock users on console do not have a built in ability to add custom servers, so they have to follow a tutorial specific to their console to change their DNS settings. Additionally, players on console need their console's subscription to give them the ability to play online (Xbox Game Pass, Switch Online, etc.).
Bedrock users, try these DNS settings. (Xbox, go to your settings app/General/Network Settings/Advanced settings/DNS Settings.)
Primary: 139.59.200.206
Secondary: 8.8.8.8
You can lookup a tutorial on YouTube, but I recommend using these DNS settings instead of their likely outdated ones. If these DNS settings do not work, please directly message me.
Attempt to connect with the primary DNS setting on CubeCraft.

Bedrock Whitelisting

Geyser/Floodgate doesn't recognize Bedrock usernames for me to whitelist until you attempt to connect. When your connection is unsuccessful, I can whitelist you. Attempt and fail to connect to the server, then directly message me with your username to be whitelisted.
